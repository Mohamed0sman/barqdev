### Step 1: Add CNAME File
Create a file named `CNAME` (no extension) with your domain:
```

```

### Step 2: Configure DNS
Add these DNS records at your domain provider:

**For www subdomain:**
```
Type: CNAME
Name: www
Value: mohamed0sman.github.io
```

**For root domain:**
